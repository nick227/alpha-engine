# Alpha Engine Architecture

```text
raw events
→ scored events
→ MRA outcomes
→ strategy runners (news + technical)
→ predictions
→ prediction outcomes
→ strategy performance ranking
```

## Tracks
- News/media track: text scoring + MRA
- Quant track: technical indicator systems using the same event + price context pipeline

## Goal
Make different systems directly comparable under one prediction and evaluation model.
